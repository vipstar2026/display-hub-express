## المرحلة 2 — Lovable Cloud + قاعدة البيانات

تفعيل Lovable Cloud وإنشاء الجداول الأساسية:

- `profiles` — بيانات المستخدمين
- `user_roles` — أدوار (buyer / seller / admin) — جدول منفصل لمنع التصعيد الأمني
- `vendors` — بيانات البائعين (اسم المتجر، الشعار، الوصف، الحالة)
- `categories` — الفئات والفئات الفرعية
- `products` — المنتجات (اسم، وصف، سعر، خصم، مخزون، vendor_id، category_id)
- `product_images` — صور متعددة لكل منتج
- `reviews` — تقييمات المنتجات

مع تفعيل RLS على كل الجداول.

## المرحلة 4 — لوحة تحكم البائع (Vendor Dashboard)

- تسجيل كبائع `/sell/register`
- لوحة `/sell/dashboard`:
  - إضافة/تعديل/حذف منتج
  - رفع صور المنتجات (Storage)
  - إدارة المخزون
  - عرض الطلبات الواردة
  - تحديث حالة الطلب (قيد المعالجة / تم الشحن / مكتمل)
- إحصائيات أساسية (مبيعات اليوم / الشهر)

## المرحلة 5 — السلة والطلبات

- جدول `cart_items` (مرتبط بالمستخدم)
- جدول `orders` و `order_items`
- جدول `addresses` (عناوين الشحن)
- إضافة للسلة من أي صفحة منتج
- صفحة Checkout (اختيار العنوان، طريقة الدفع، ملخص الطلب)
- إنشاء الطلب وتقسيمه على البائعين تلقائياً

---

## المرحلة 6 — الدفع (Payment)

- ربط Stripe (يدعم بطاقات + Apple Pay + Google Pay)
- دعم الدفع عند الاستلام (Cash on Delivery) كخيار
- Webhook لتأكيد الدفع وتحديث حالة الطلب
- إرسال بريد تأكيد الطلب (Lovable Email)

---

## المرحلة 7 — التقييمات والبحث المتقدم

- نظام تقييمات (1-5 نجوم + تعليق) — بعد الشراء فقط
- بحث متقدم بالـ Full Text Search
- فلاتر متعددة (سعر، تقييم، علامة، توفر)
- ترتيب (الأحدث، الأقل سعراً، الأعلى تقييماً، الأكثر مبيعاً)

---

## المرحلة 8 — لوحة الأدمن والتشطيب

- لوحة `/admin` (للمالك فقط):
  - الموافقة على البائعين الجدد
  - إدارة الفئات
  - حذف منتجات/تعليقات مخالفة
  - إحصائيات الموقع الكلية
- نظام إشعارات (طلب جديد، تقييم جديد)
- صفحة 404 مخصصة + SEO لكل صفحة
- اختبار شامل + نشر

---

## التفاصيل التقنية

- **Routing**: TanStack Start (موجود)
- **Database & Auth & Storage**: Lovable Cloud (Supabase تحت الغطاء)
- **Payment**: Stripe via Lovable
- **Email**: Lovable Email للتأكيدات
- **State**: TanStack Query للبيانات
- **i18n**: نظام موجود حالياً، سنوسّعه لباقي الصفحات
