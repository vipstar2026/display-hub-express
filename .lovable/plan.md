
# خطة إعادة بناء VIPSTAR من الصفر

## نظرة عامة
متجر إلكتروني احترافي متخصص في:
- 📡 أجهزة ومستلزمات الستلايت (رسيفرات، أطباق، LNB، كوابل)
- 📺 اشتراكات IPTV (منتجات رقمية)
- 🛠️ إكسسوارات ومستلزمات

**التصميم:** داكن تقني (Cyberpunk) — أسود عميق `#0a0a0f`, سيان نيون `#00d9ff`
**اللغات:** عربي (افتراضي RTL) / إنجليزي / أردو
**النشر:** Lovable Cloud + لاحقاً VPS (vipstar.cc)

---

## 1. تنظيف شامل (مرحلة الهدم)

- حذف كل جداول قاعدة البيانات الحالية (products, vendors, orders, ...)
- حذف bucket `vendor-assets`
- حذف كل ملفات `src/routes/_authenticated/admin.*` القديمة
- حذف ملفات `scripts/vps-setup.sh` و workflows القديمة
- الاحتفاظ فقط بالبنية الأساسية: `__root.tsx`, `index.tsx`, auth integration

---

## 2. قاعدة البيانات الجديدة (Migration واحد)

**جداول جديدة نظيفة:**

| الجدول | الغرض |
|---|---|
| `profiles` | بيانات المستخدم (اسم، صورة، هاتف) |
| `user_roles` | الأدوار (admin, customer) — جدول منفصل آمن |
| `categories` | الفئات (رسيفرات، أطباق، IPTV، ...) مع slug ثلاثي اللغة |
| `products` | المنتجات (مادي/رقمي، سعر، مخزون، صور، وصف 3 لغات) |
| `product_variants` | اختياري: مدة الاشتراك IPTV (شهر/3شهور/سنة) |
| `iptv_codes` | أكواد التفعيل الرقمية (تُسلَّم تلقائياً بعد الدفع) |
| `addresses` | عناوين الشحن |
| `orders` + `order_items` | الطلبات |
| `payments` | سجلات Stripe webhook |
| `coupons` | كوبونات الخصم |
| `reviews` | تقييمات المنتجات |
| `site_settings` | الإعدادات العامة (شعار، روابط، عملة) |

كل الجداول مع RLS + GRANT صحيحين من البداية.

---

## 3. الواجهة الأمامية

```
src/routes/
├── index.tsx                    # واجهة المتجر (Hero + فئات + أحدث المنتجات)
├── shop.tsx + shop.index.tsx    # تصفح المنتجات (فلاتر، بحث)
├── product.$slug.tsx            # صفحة منتج
├── cart.tsx                     # السلة
├── checkout.tsx                 # الدفع (Stripe Checkout)
├── orders.success.tsx           # تأكيد الطلب
├── about.tsx, contact.tsx
├── auth.tsx                     # دخول/تسجيل (Email + Google)
└── _authenticated/
    ├── account.tsx              # حساب المستخدم
    ├── account.orders.tsx       # طلباتي + أكواد IPTV
    └── admin.*                  # لوحة تحكم إدارية كاملة
```

**لوحة التحكم تشمل:**
- داشبورد (إيرادات، طلبات، عملاء)
- إدارة المنتجات + الفئات + المخزون
- إدارة الطلبات + الحالات + تتبع الشحن
- مخزون أكواد IPTV
- إدارة المستخدمين + الأدوار
- الكوبونات + الإعدادات
- التقارير

---

## 4. المدفوعات

**Stripe** (built-in من Lovable) — يدعم بطاقات + Apple Pay + Google Pay
- Webhook لمعالجة `checkout.session.completed`
- تسليم تلقائي لأكواد IPTV بعد نجاح الدفع
- إيميل تأكيد للعميل

---

## 5. تعدد اللغات

- مكتبة `i18next` مع كشف اللغة من المتصفح
- 3 ملفات ترجمة (ar.json, en.json, ur.json)
- التبديل من Header
- RTL تلقائي للعربي/الأردو، LTR للإنجليزي

---

## 6. النشر

**المرحلة 1 (فورية):** Lovable Cloud → vipstar.cc يعمل مباشرة
**المرحلة 2 (لاحقاً):** GitHub Actions → VPS بطريقة احترافية (PM2 + Nginx + Wrangler)

---

## ⚠️ تحذيرات مهمة

1. **لا رجعة:** كل البيانات الحالية ستُحذف نهائياً
2. **الوقت:** البناء سيتم على دفعات (Migration → جداول → واجهة → لوحة تحكم → دفع)
3. **VPS:** بعد ما الموقع يستقر على Lovable، نتجه للـ VPS
